# Design Diff Checker (デザイン差分確認ツール)

ブラウザひとつで完結する、セキュアで高機能な画像差分比較ツールです。
デザインカンプと実装画面のズレ確認や、修正前後の画像比較に最適です。

![Screenshot](https://images.unsplash.com/photo-1611162617474-5b21e879e113?q=80&w=1000&auto=format&fit=crop) 
*(※スクリーンショットのイメージ)*

## 🚀 特徴

- **インストール不要**: `index.html` を開くだけですぐに使えます。
- **完全セキュア**: 画像データはサーバーに送信されず、すべてお使いのブラウザ内（ローカル）で処理されます。
- **3つの比較モード**:
    1.  **Diff (差分検知)**: 異なる箇所を赤くハイライト表示します。
    2.  **Overlay (重ね合わせ)**: 2枚の画像を重ね、透明度を調整して比較できます（オニオンスキン）。
    3.  **Side-by-Side (並列)**: 画像を横に並べて見比べます。
- **微調整機能**: ノイズ除去のための「しきい値(Threshold)」設定や、重ね合わせ時の「透明度(Opacity)」調整が可能です。

## 📖 使い方

1.  **起動する**
    - フォルダ内の `index.html` をダブルクリックして、ChromeやEdgeなどのブラウザで開いてください。

2.  **画像を読み込む**
    - 左側のエリアに **「比較元（Before）」** の画像をドラッグ＆ドロップします。
    - 右側のエリアに **「比較先（After）」** の画像をドラッグ＆ドロップします。
    - *※クリックしてファイル選択画面から選ぶことも可能です。*

3.  **比較する**
    - 画像が2枚揃うと、自動的に比較画面が表示されます。
    - 画面上部のボタンでモードを切り替えてください。
        - **Diff**: 差分がある場所が「赤色」で表示されます。スライダーで感度を調整できます。
        - **Overlay**: スライダーを左右に動かして、画像の重なり具合を確認できます。

## ⚠️ 注意事項

- **画像サイズについて**: サイズが異なる画像同士も比較可能ですが、その場合「左上合わせ」で比較されます。
- **対応ブラウザ**: 最新の Google Chrome, Microsoft Edge, Firefox, Safari での動作を推奨します。
