# Design Diff Checker (デザイン差分確認ツール)

ブラウザひとつで完結する、セキュアで高機能な画像差分比較ツールです。
デザインカンプと実装画面のズレ確認や、修正前後の画像比較に最適です。



## 🚀 特徴

- **インストール不要**: `index.html` を開くだけですぐに使えます。
- **完全セキュア**: 画像データはサーバーに送信されず、すべてお使いのブラウザ内（ローカル）で処理されます。
- **3つの比較モード**:
    1.  **Diff (差分検知)**: 異なる箇所を赤くハイライト表示します。
    2.  **Overlay (重ね合わせ)**: 2枚の画像を重ね、透明度を調整して比較できます（オニオンスキン）。
